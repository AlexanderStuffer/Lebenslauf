# Lebenslauf-Builder

Lokales Tool zum Erstellen eines Lebenslaufs mit:
- Eingabeformular für persönliche Daten
- Live-Vorschau während der Bearbeitung
- Design-Anpassungen (Farben, Layout, Schrift, Formen, Abstände)
- Elementweises Feintuning (pro Block X/Y, Breite, Schriftgröße, Zeilenhöhe)
- Optionales Drag-and-drop für einzelne Lebenslauf-Elemente direkt in der Vorschau
- Profilbild-Upload und Projektbild-Upload mit vorgefertigten Rahmen
- Zuschnitt-/Positionierungsoptionen für Bilder (Zoom, X/Y)
- Aktivierbare/deaktivierbare CV-Elemente mit automatischer Neuanordnung
- Bereiche für Berufserfahrung, Praktika, Ausbildung, Sprachen, Kenntnisse (Skala), Zertifizierungen und Projekte
- Projekte-Bereich mit optionalen Feldern (Kunde, Rolle, Status, Links, Highlights)
- Kategorien (optional) mit frei wählbarer Position (Header, Sidebar, Hauptbereich)
- JSON-Export/Import zum Speichern und späteren Weiterbearbeiten des kompletten Zustands
- PDF-Download per Button

## Start

1. Datei `index.html` im Browser öffnen.
2. Daten links eintragen.
3. Vorschau rechts live prüfen.
4. Auf `Als PDF herunterladen` klicken.

## Hinweise

- Für den PDF-Export wird `html2pdf.js` von einem CDN geladen. Dafür ist eine Internetverbindung nötig.
- Wenn du schnell ein Beispiel sehen willst, klicke auf `Demo-Daten laden`.
